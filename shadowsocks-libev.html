shadowsocks-libev安装使用 简单明了

linux

网上抄来抄去 垃圾信息太多 本来很简单的事情 不知道为什么都说的那么复杂 官网也给一堆垃圾信息 之前放在阿里云，这篇文章竟然被屏蔽了。

 cd /etc/yum.repos.d/
 wget https://copr.fedorainfracloud.org/coprs/librehat/shadowsocks/repo/epel-7/librehat-shadowsocks-epel-7.repo
 yum install shadowsocks-libev

编辑配置文件

{   
 "server":["[::0]","0.0.0.0"],#容许IPv6和4都访问  
  "server_port":8080, 
   "local_port":1080, 
   "password":"xxx",  
  "timeout":60,  
  "method":"aes-128-cfb"  #加密算法自己选}

开启服务

 /etc/init.d/shadowsocks-libev start

查看状态

/etc/init.d/shadowsocks-libev status
