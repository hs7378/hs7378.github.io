ACS UCP（User Change Password）自主修改账号密码页面

思科

1.在ACS上下载文件

System Administration > Downloads > User Change Password > UCP web application 下载后

2.开启UCP ACS web interface

acs5-8/admin# acs config-web-interface ucp enable                
acs5-8/admin# show acs-config-web-interface 
migration interface is disabled
ucp interface is enabled
view interface is disabled

3.搭建jsp环境

搭建jsp环境，并将文件拷入网站根目录，必须是根目录，子目录就会报错。 UCP需要一个jsp的web环境，一般用Tomcat即可，以下是简单的搭建过程:

需要下载JDK和Tomcat。先安装JDK，添加系统环境变量 名称：JAVA_HOME 变量值：安装的路径（例如C:\Program Files\Java\jdk1.8.0_151） 再添加名称：CLASS_PATH 变量值：.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar 注意符号 再再PATH变量里面增加 ;%JAVA_HOME%\bin; Tomcat安装后也需要添加环境变量 变量名：CATALINA_HOME 变量值：安装的路径 http://www.ciscopress.com/articles/article.asp?p=1678922
